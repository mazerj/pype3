Module movie

source code

Sprite-based movie classes

These classes provide bare-bones movie classes. Movies are stored as individual frames in a directory. Each movie directory must contain an indexfile (defaultname is INDEX) that lists each individual frame mage file, in playback order.

These classes are primarily provided as examples for how to handle large image sets in pype. There are probably more efficient ways to anything you might be doing than using these classes unmodified.
